# Hotel Reservation Management System (DBMS Project)

This project presents the database design for a Hotel Reservation Management System.  
It was developed as part of the Database Systems course.

## Project Objectives

The goal of this project is to design and implement a structured database that manages hotel operations including:

- Room reservations
- Customer information
- Booking management
- Payment tracking

## Database Design

The database was designed using multiple steps:

1. Entity Relationship (ER) Diagram
2. Enhanced Entity Relationship (EER) Model
3. Relational Schema Conversion
4. Database Normalization

## Project Components

- ER Diagram
- EER Diagram
- Relational Schema
- Normalization Process
- SQL Queries
- Project Proposal
- Project Presentation

- SQL
